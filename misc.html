Miscellaneous

Opinions on Notation, Definitions, and Mathematical Writing

  • This symbol, \(\subset\), means subset. This symbol, \(\subsetneq\), means proper subset.
  • When defining limits with epsilon-delta language, it is \(0 < |x - a| < \delta\), and not \(|x - a| < \delta\) (as done in France).
  • Avoid unnecessary use of logic symbols like \(\forall, \exists, \land, \lor, \neg, \therefore\). Natural language is more readable for everyone.
  • Writing integrals like \(\int_\mathbb{R}f(x)dx\) is preferable to writing them like \(\int_{-\infty}^\infty f(x)dx\).
  • It is okay to write sums of lets like \(\sum_A\). However, when talking about series, do not write something like \(\sum_\mathbb{N}\)
  • Leibniz's notation for differentiation (\(\frac{df}{dx}, \frac{\partial f}{\partial x}, etc\) ) is preferable.
  • While sometimes inconvenient, it must be that \(0 \in \mathbb{N} \) because of the set-theoretic definition of the natural numbers.
  • Write vectors like \(\mathbf{u, v, w}\), and not like \(\vec{u}, \vec{v}, \vec{w}\).
  • The transpose of a matrix is denoted \(A^\top\).
  • Use \(\mathcal{T}\) for topologies and linear transformations , as opposed to \(\tau\)
  • Unless working in a space that is not path-connected, feel free to omit the base point when writing about homotopy groups.
  • Do not use \(\hookrightarrow, \twoheadrightarrow, \longleftrightarrow\)
  • Write group presentations like \(\langle a,b \text{ } | \text{ } a = b\rangle\) as opposed to \(\langle a,b \text{ } | \text{ } ab^{-1} \rangle \).
